This is a [Next.js](https://nextjs.org) project bootstrapped with [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).

## Getting Started

- Abra a pasta do projeto com o VSCode;

- Instale a extensão "Dev Containers", do VSCode;

- Utilize o comando "Dev Containers: Rebuild and Reopen Container Without Cache";

- Renomeie example.env para .env;

- Utilize o comando
```bash
pnpm run dev
```
no bash para inicializar o servidor.

- Acesse o servidor em [http://localhost:3000].
